# Campus Info Hub

1. **Ce este o resursa (resource)?**
   O resursa reprezinta un loc din campus (ex: biblioteca, cantina, sala de evenimente) cu informatii despre locatie, program si tip.

2. **Exemplu URI:**
   `/pages/library.html#schedule`
   - `/pages/library.html` -> fisierul HTML
   - `#schedule` -> fragment link catre sectiunea orar

3. **Parti statice si dinamice:**
   - Statice: HTML-ul de baza, CSS-ul
   - Dinamice: Lista resurselor incarcata din JSON prin JavaScript

4. **Aplicatia este document-centric sau interactive?**
   Este interactiva, deoarece foloseste JavaScript pentru a incarca si filtra resurse din JSON in mod dinamic.
