# Campus Info Hub

## 1. Ce este o resursa in aplicatia mea?

O resursa este orice serviciu sau locatie din campus care poate fi accesata de studenti. Exemple: biblioteca, cantina sau sala de evenimente.

## 2. Exemplu de URI

Exemplu:
pages/library.html#schedule

Componente:
- pages = director
- library.html = fisier
- #schedule = fragment care duce la sectiunea programului

## 3. Parti statice si dinamice

Statice:
- paginile HTML
- structura site-ului

Dinamice:
- datele incarcate din resources.json
- lista afisata cu JavaScript folosind fetch()

## 4. Aplicatia este document-centric sau interactive?

Aplicatia este ambele:
- document-centric deoarece prezinta informatii despre resurse
- interactive deoarece foloseste JavaScript pentru a afisa si filtra datele.
